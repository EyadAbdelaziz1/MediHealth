# Medi Health 🏥

Medi Health is an open-source healthcare platform focused on making health management simpler, more accessible, and more organized for everyone.

## Features

* Patient management
* Appointment scheduling
* Medication reminders
* Medical records management
* Secure authentication
* AI-powered health assistance
* Mobile-friendly experience

## Vision

Healthcare technology should be accessible to everyone. Medi Health aims to provide a modern, open-source platform that helps users manage their health information while giving developers a foundation to build innovative healthcare solutions.

## Tech Stack

* Next.js
* React
* TypeScript
* Node.js
* Appwrite
